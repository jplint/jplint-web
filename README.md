- Vue@3

## How to

### Setup

```shell
npm install
```

### Develop

#### Compiles and hot-reloads for development

```shell
npm run serve
```

#### Compiles and minifies for production

```shell
npm run build
```

#### Lints and fixes files

```shell
npm run lint
```

### Test

#### Unit Test

```shell
npm run test:unit
```

#### E2E Test

```shell
npm run test:e2e
```
